# miniz tinfl

This directory vendors the low-level `tinfl` decompressor from the official
miniz repository for Catime's embedded zlib/Deflate resources. It does not use
a system compression DLL and does not include miniz's compressor or ZIP APIs.

- Upstream: https://github.com/richgel999/miniz
- Version: 3.1.2
- Commit: `77d0dce8627735138c51770d1799a1ef48f2117d`
- License: MIT; see `LICENSE`

Vendored upstream files are `miniz_tinfl.c`, `miniz_tinfl.h`, and
`miniz_common.h`. `miniz_export.h` was generated by the upstream CMake build
at the pinned commit for a static library.

The only source adaptation is in `miniz_tinfl.c`: its umbrella include
`miniz.h` is replaced with `miniz_tinfl.h`. This keeps the dependency limited
to tinfl without changing the decompression implementation.

Compile both the library source and its consumers with `MINIZ_NO_STDIO`,
`MINIZ_NO_TIME`, and `MINIZ_NO_MALLOC`. Catime supplies all input, output, and
decompressor storage to the low-level `tinfl_decompress()` API.
