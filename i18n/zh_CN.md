## 🎨 强大的托盘动画

你的托盘图标现在可以“活”起来了！Catime 带来了强大的自定义动画能力：

- **自定义动画**：支持将 **GIF**、**WebP** 以及 **PNG**、**JPG** 等静态图片作为托盘图标。
- **系统状态显示**：支持实时显示 **CPU 占用**、**内存占用**、**电量百分比** 与 **Caps Lock** 状态。

## [📑 使用指南](https://cati.me/guide)

- **左键点击托盘图标**：设置 **时间**
- **右键点击托盘图标**：打开 **菜单**
- **编辑模式**：拖动可调整位置，滚轮可缩放，🖱️ 右键可退出编辑模式

> 配置文件保存在 `%LOCALAPPDATA%\Catime\config.ini`

## ⚡ [字体优化](https://vladelaina.github.io/Catime/tools/font-tool/)

使用官方 [字体精简工具](https://vladelaina.github.io/Catime/tools/font-tool/) 可将自定义字体体积缩小 **90% 以上**。

> 由于 Catime 主要显示数字和符号（`0-9`、`:`、`.`），大部分字体字符其实并不需要。
>
> **建议**：请保留原始字体文件作为备份。后续软件更新可能会需要更多字符。

## 🛠️ 开发

**前置要求**：构建 Catime 需要 MinGW 和 CMake。

## 开始使用

```bash
git clone https://github.com/vladelaina/Catime.git
cd Catime

# 安装依赖
# Ubuntu/Debian
sudo apt update && sudo apt install -y mingw-w64 cmake

# Arch Linux
sudo pacman -Syu --noconfirm mingw-w64 cmake

# Windows - 见下方配置说明
```

<details>
<summary>📋 Windows 环境配置说明</summary>

### 📦 必备工具

| 工具 | 说明 | 推荐版本格式 | 下载链接 |
| ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **MinGW**  | GCC 编译器 | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) |
| **CMake**  | 构建工具 | `cmake-<version>-windows-x86_64.msi` | [CMake](https://cmake.org/download/) |

### 🔧 安装步骤

#### 1. 安装 MinGW
1. 从 [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) 下载 `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z`
2. 解压到 `C:\mingw64`
3. 将 `C:\mingw64\bin` 添加到系统 `PATH` 环境变量：
   - 打开：`控制面板 → 系统 → 高级系统设置 → 环境变量`
   - 在 **系统变量** 中找到 `Path`，点击“编辑”
   - 添加：`C:\mingw64\bin`

#### 2. 安装 CMake
1. 从 [CMake](https://cmake.org/download/) 下载并运行 `cmake-<version>-windows-x86_64.msi`
2. 安装过程中选择 “Add CMake to the system PATH”

#### 3. 验证安装
```bash
gcc --version
cmake --version
```

</details>
