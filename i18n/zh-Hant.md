## 🎨 強大的托盤動畫

你的托盤圖標現在可以「活」起來了！Catime 帶來了強大的自定義動畫能力：

- **自定義動畫**：支持將 **GIF**、**WebP** 以及 **PNG**、**JPG** 等靜態圖片作為托盤圖標。
- **系統狀態顯示**：支持實時顯示 **CPU 佔用**、**記憶體佔用**、**電量百分比** 與 **Caps Lock** 狀態。

## [📑 使用指南](https://cati.me/guide)

- **左鍵點擊托盤圖標**：設置 **時間**
- **右鍵點擊托盤圖標**：打開 **選單**
- **編輯模式**：拖動可調整位置，滾輪可縮放，🖱️ 右鍵可退出編輯模式

> 配置文件保存在 `%LOCALAPPDATA%\Catime\config.ini`

## ⚡ [字體優化](https://vladelaina.github.io/Catime/tools/font-tool/)

使用官方 [字體精簡工具](https://vladelaina.github.io/Catime/tools/font-tool/) 可將自定義字體體積縮小 **90% 以上**。

> 由於 Catime 主要顯示數字和符號（`0-9`、`:`、`.`），大部分字體字符其實並不需要。
>
> **建議**：請保留原始字體文件作為備份。後續軟體更新可能會需要更多字符。

## 🛠️ 開發

**前置要求**：構建 Catime 需要 MinGW 和 CMake。

## 開始使用

```bash
git clone https://github.com/vladelaina/Catime.git
cd Catime

# 安裝依賴
# Ubuntu/Debian
sudo apt update && sudo apt install -y mingw-w64 cmake

# Arch Linux
sudo pacman -Syu --noconfirm mingw-w64 cmake

# Windows - 見下方配置說明
```

<details>
<summary>📋 Windows 環境配置說明</summary>

### 📦 必備工具

| 工具 | 說明 | 推薦版本格式 | 下載鏈接 |
| ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **MinGW** | GCC 編譯器 | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) |
| **CMake** | 構建工具 | `cmake-<version>-windows-x86_64.msi` | [CMake](https://cmake.org/download/) |

### 🔧 安裝步驟

#### 1. 安裝 MinGW
1. 從 [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) 下載 `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z`
2. 解壓到 `C:\mingw64`
3. 將 `C:\mingw64\bin` 添加到系統 `PATH` 環境變量：
   - 打開：`控制台 → 系統 → 進階系統設定 → 環境變量`
   - 在 **系統變量** 中找到 `Path`，點擊「編輯」
   - 添加：`C:\mingw64\bin`

#### 2. 安裝 CMake
1. 從 [CMake](https://cmake.org/download/) 下載並運行 `cmake-<version>-windows-x86_64.msi`
2. 安裝過程中選擇 “Add CMake to the system PATH”

#### 3. 驗證安裝
```bash
gcc --version
cmake --version
```

</details>
