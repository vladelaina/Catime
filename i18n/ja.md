## 🎨 強力なトレイアニメーション

トレイアイコンが今、「生き生き」と動き出します！Catimeは強力なカスタムアニメーション機能を提供します：

- **カスタムアニメーション**：**GIF**、**WebP**、および **PNG**、**JPG** などの静的画像をトレイアイコンとして使用できます。
- **システム状態の表示**：**CPU使用率**、**メモリ使用量**、**バッテリー残量（％）**、そして **Caps Lock** の状態をリアルタイム表示します。

## [📑 使い方ガイド](https://cati.me/guide)

- **トレイアイコンの左クリック**：**時間** を設定
- **トレイアイコンの右クリック**：**メニュー** を開く
- **編集モード**：ドラッグで位置を調整、マウスホイールでズーム、🖱️ の右クリックで編集モードを終了

> 設定ファイルは `%LOCALAPPDATA%\Catime\config.ini` に保存されます

## ⚡ [フォント最適化](https://vladelaina.github.io/Catime/tools/font-tool/)

公式の [フォント圧縮ツール](https://vladelaina.github.io/Catime/tools/font-tool/) を使うことで、カスタムフォントのサイズを **90% 以上** 縮小できます。

> Catimeは主に数字と記号（`0-9`、`:`、`.`）を表示するため、ほとんどのフォント文字は実際には必要ありません。
>
> **おすすめ**：バックアップとして元のフォントファイルを残しておいてください。今後のソフトウェア更新で、より多くの文字が必要になる可能性があります。

## 🛠️ 開発

**前提条件**：CatimeのビルドにはMinGWとCMakeが必要です。

## はじめに

```bash
git clone https://github.com/vladelaina/Catime.git
cd Catime

# 依存関係をインストール
# Ubuntu/Debian
sudo apt update && sudo apt install -y mingw-w64 cmake

# Arch Linux
sudo pacman -Syu --noconfirm mingw-w64 cmake

# Windows - 下の設定説明を参照
```

<details>
<summary>📋 Windows環境の設定説明</summary>

### 📦 必須ツール

| ツール | 説明 | 推奨バージョン形式 | ダウンロードリンク |
| ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **MinGW**  | GCC コンパイラ | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) |
| **CMake**  | ビルドツール | `cmake-<version>-windows-x86_64.msi` | [CMake](https://cmake.org/download/) |

### 🔧 インストール手順

#### 1. MinGWをインストール
1. [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) から `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` をダウンロード
2. `C:\mingw64` に解凍
3. `C:\mingw64\bin` をシステムの `PATH` 環境変数に追加：
   - 開く：`コントロール パネル → システム → システムの詳細設定 → 環境変数`
   - **システム変数** の中から `Path` を見つけて「編集」
   - 追加：`C:\mingw64\bin`

#### 2. CMakeをインストール
1. [CMake](https://cmake.org/download/) から `cmake-<version>-windows-x86_64.msi` をダウンロードして実行
2. インストール中に「Add CMake to the system PATH」を選択

#### 3. インストールを確認
```bash
gcc --version
cmake --version
```

</details>
