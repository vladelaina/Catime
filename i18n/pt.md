## 🎨 Animacoes fortes da bandeja

O seu icone da bandeja agora pode "ganhar vida"! O Catime traz capacidades poderosas para animacao personalizada:

- **Animacoes personalizadas**: suporta usar **GIF**, **WebP** e imagens estaticas como **PNG** e **JPG** como icones da bandeja.
- **Exibicao do estado do sistema**: suporta exibir em tempo real **uso de CPU**, **uso de memoria**, **percentual de bateria** e o status de **Caps Lock**.

## [📑 Guia de uso](https://cati.me/guide)

- **Clique esquerdo no icone da bandeja**：Defina **Tempo**
- **Clique direito no icone da bandeja**：Abra **Menu**
- **Modo de edicao**：Arraste para ajustar a posicao, roda do mouse para dar zoom, clique direito com 🖱️ para sair do modo de edicao

> O arquivo de configuracao e salvo em `%LOCALAPPDATA%\Catime\config.ini`

## ⚡ [Otimizacao de fontes](https://vladelaina.github.io/Catime/tools/font-tool/)

Use a ferramenta oficial [Ferramenta de reducao de fontes](https://vladelaina.github.io/Catime/tools/font-tool/) para reduzir o tamanho das suas fontes personalizadas em **mais de 90%**.

> Como o Catime mostra principalmente numeros e simbolos (`0-9`、`:`、`.`), a maioria dos caracteres da fonte nao e, na verdade, necessaria.
>
> **Recomendacao**: Mantenha os arquivos originais da fonte como backup. Futuras atualizacoes do software podem exigir mais caracteres.

## 🛠️ Desenvolvimento

**Requisitos**: Para compilar o Catime, e necessario MinGW e CMake.

## Comecar

```bash
git clone https://github.com/vladelaina/Catime.git
cd Catime

# Instalar dependencias
# Ubuntu/Debian
sudo apt update && sudo apt install -y mingw-w64 cmake

# Arch Linux
sudo pacman -Syu --noconfirm mingw-w64 cmake

# Windows - ver as notas de configuracao abaixo
```

<details>
<summary>📋 Instrucoes de configuracao do ambiente Windows</summary>

### 📦 Ferramentas indispensaveis

| Ferramenta | Descricao | Formato recomendado da versao | Link de download |
| ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **MinGW**  | Compilador GCC | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) |
| **CMake**  | Ferramenta de build | `cmake-<version>-windows-x86_64.msi` | [CMake](https://cmake.org/download/) |

### 🔧 Passos de instalacao

#### 1. Instalar MinGW
1. Baixe `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` em [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)
2. Extraia para `C:\mingw64`
3. Adicione `C:\mingw64\bin` a variavel de ambiente do sistema `PATH`:
   - Abrir: `Painel de controle → Sistema → Configuracoes avancadas do sistema → Variaveis de ambiente`
   - Em **variaveis do sistema**, encontre `Path` e clique em "Editar"
   - Adicionar: `C:\mingw64\bin`

#### 2. Instalar CMake
1. Baixe e execute `cmake-<version>-windows-x86_64.msi` em [CMake](https://cmake.org/download/)
2. Durante a instalacao, selecione "Add CMake to the system PATH"

#### 3. Verificar a instalacao
```bash
gcc --version
cmake --version
```

</details>
