## 🎨 강력한 트레이 애니메이션

이제 트레이 아이콘이 “살아 움직일” 수 있습니다! Catime은 강력한 사용자 정의 애니메이션 기능을 제공합니다:

- **사용자 정의 애니메이션**: **GIF**, **WebP** 및 **PNG**, **JPG** 같은 정적 이미지를 트레이 아이콘으로 사용할 수 있습니다.
- **시스템 상태 표시**: **CPU 사용량**, **메모리 사용량**, **배터리 비율(%)** 과 **Caps Lock** 상태를 실시간으로 표시합니다.

## [📑 사용 가이드](https://cati.me/guide)

- **트레이 아이콘 왼쪽 클릭**：**시간** 설정
- **트레이 아이콘 오른쪽 클릭**：**메뉴** 열기
- **편집 모드**：드래그로 위치 조정, 마우스 휠로 확대/축소, 🖱️ 오른쪽 클릭으로 편집 모드 종료

> 설정 파일은 `%LOCALAPPDATA%\Catime\config.ini`에 저장됩니다

## ⚡ [글꼴 최적화](https://vladelaina.github.io/Catime/tools/font-tool/)

공식 [글꼴 축소 도구](https://vladelaina.github.io/Catime/tools/font-tool/)를 사용하면 사용자 지정 글꼴 크기를 **90% 이상** 줄일 수 있습니다.

> Catime은 주로 숫자와 기호(`0-9`、`:`、`.`)를 표시하므로, 대부분의 글꼴 문자는 실제로 필요하지 않습니다.
>
> **권장**: 원본 글꼴 파일을 백업으로 보관해 주세요. 향후 소프트웨어 업데이트에서 더 많은 문자가 필요할 수 있습니다.

## 🛠️ 개발

**선행 조건**: Catime을 빌드하려면 MinGW와 CMake가 필요합니다.

## 시작하기

```bash
git clone https://github.com/vladelaina/Catime.git
cd Catime

# 의존성 설치
# Ubuntu/Debian
sudo apt update && sudo apt install -y mingw-w64 cmake

# Arch Linux
sudo pacman -Syu --noconfirm mingw-w64 cmake

# Windows - 아래 설정 설명을 참고하세요
```

<details>
<summary>📋 Windows 환경 설정 설명</summary>

### 📦 필수 도구

| 도구 | 설명 | 권장 버전 형식 | 다운로드 링크 |
| ---------- | ---------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **MinGW**  | GCC 컴파일러 | `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` | [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest) |
| **CMake**  | 빌드 도구 | `cmake-<version>-windows-x86_64.msi` | [CMake](https://cmake.org/download/) |

### 🔧 설치 단계

#### 1. MinGW 설치
1. [MinGW Build](https://github.com/niXman/mingw-builds-binaries/releases/latest)에서 `x86_64-<version>-release-win32-seh-ucrt-rtv<num>-rev<num>.7z` 다운로드
2. `C:\mingw64`로 압축 해제
3. `C:\mingw64\bin`을 시스템 `PATH` 환경 변수에 추가:
   - 열기: `제어판 → 시스템 → 고급 시스템 설정 → 환경 변수`
   - **시스템 변수**에서 `Path`를 찾아 "편집"을 클릭
   - 추가: `C:\mingw64\bin`

#### 2. CMake 설치
1. [CMake](https://cmake.org/download/)에서 `cmake-<version>-windows-x86_64.msi`를 다운로드하고 실행
2. 설치 과정에서 "Add CMake to the system PATH"를 선택

#### 3. 설치 확인
```bash
gcc --version
cmake --version
```

</details>
